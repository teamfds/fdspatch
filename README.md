Dependencias:
`python3 cargo rustc`

build.py compila o mod  
build.py -I compila o mod e instala na pasta do jogo definida em .env\[GAME_PATH\]  
build_tools.py compila as ferramentas :))  
tools/fix_version.py corrige a versao de uassets de patches anteriores  
tools/import_asset.py importa, e corrige, um par de uasset e uexp de res pra dentro de src  
ex: `python3 tools/import_asset.py BBS_TST_407`  
tools/parse_bbs.py extrai o bbscript em forma de texto   
ex: `python3 tools/parse_bbs.py BBS_TST_407`
